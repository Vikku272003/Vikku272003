## 🚀 Featured Projects

### 🏥 1. [NHS RTT Waiting Times Analysis](https://github.com/Vikku272003/nhs-waiting-times)
> End-to-end public health analytics · 228 weeks of real NHS England WLMDS data · Interactive live dashboard

- Only **60.7%** of patients seen within 18 weeks vs the **92% NHS target** — a 31.3pp gap
- Waiting list peaked at **7.8 million patients** in 2024, with 300K+ waiting 52+ weeks
- Found a **−0.81 correlation** between list size and 18-week performance via statistical modelling
- Forecast confirms: **92% target will not be reached** in the next 12 months
- Built an interactive **HTML/JS dashboard** from scratch with regression forecasting

`Python` `Pandas` `Scikit-learn` `SciPy` `SQL` `Linear Regression` `NHS Open Data`

🔗 [Live Dashboard](https://vikku272003.github.io/nhs-waiting-times/dashboard.html) · [GitHub Repo](https://github.com/Vikku272003/nhs-waiting-times)
